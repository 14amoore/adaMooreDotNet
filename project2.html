breadBot

breadBot is a social media bot operating on Twitter. With all of the news regarding the manipulation of the electorate via social media I wanted to see just how hard it would be to create my own social media bot. Because the topic of social media can be somewhat dark I wanted to inject a little levity into the project by having the bot tweet about bread rather then politics. I chose the topic of bread because of my favorite meme of 2018, let's get this bread.

breadBot is programmed to tweet about bread once an hour. The data was scraped from a Wikipedia page. This data was then converted into a JSON file and loaded onto an amazon server along with a JS file that contained the code for the bot itself. Click here to see breadBot's latest tweets.
